# projeto-banco-de-dados-veterinaria

Tema: Sistema de Gestão de agendamentos para Clínicas Veterinárias

Descrição do Tema: Este trabalho propõe a modelagem conceitual de um banco de dados para um sistema de gestão de clínica veterinária, considerando o crescimento do mercado pet e a maior demanda por serviços especializados.
Muitas clínicas ainda utilizam métodos pouco estruturados para organizar informações como cadastro de clientes, histórico dos animais e agendamentos, o que pode gerar falhas operacionais e dificuldade na gestão. Nesse contexto, o tema foi escolhido por sua relevância prática e pela necessidade de melhorar a organização e o controle desses processos.
O sistema proposto tem como objetivo centralizar e estruturar os dados relacionados aos atendimentos, incluindo clientes (tutores), pets, veterinários, consultas, serviços e agendamentos, além de um registro básico do histórico dos animais.
